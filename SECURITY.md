# Security Policy

## Supported Versions

Terminally takes security seriously. We currently support the following versions for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

If you identify a security vulnerability in **Terminally**, please report it responsibly by following these steps:

1. **Do not** report security vulnerabilities through public GitHub issues.
2. Email us at `security@terminally.ai` or contact the maintainers directly through Discord.
3. Provide as much detail as possible, including steps to reproduce and potential impact.

## Our Response

- We will acknowledge receipt of your report within **48 hours**.
- We will work to investigate and address the issue within **30 days**.
- You will receive credit for the discovery in our Security Updates.

## Scope

This policy applies only to the core **Terminally** CLI tool and its official components. Third-party integrations or unofficial plugins are not covered.
